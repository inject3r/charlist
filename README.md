# CharList

A collection of characters, symbols, emojis, hieroglyphs, and special characters. Click to copy.

## Live Demo

[https://inject3r.github.io/charlist/](https://inject3r.github.io/charlist/)

## Screenshot

![CharList Screenshot](https://raw.githubusercontent.com/inject3r/charlist/master/screenshot.jpg)

## Features

- 6000+ characters across 10 categories
- Search by character or name
- Recent copies stored locally (last 6)
- Click to copy with toast notification
- Responsive dark theme
- Auto pagination for large lists

## Categories

Popular, Emoji, Arrows, Hieroglyphs, Symbols, Currency, Numbers, Math, Punctuation, Letters

## Tech Stack

React 19, Vite, Tailwind CSS, Lucide React

## Quick Start

```bash
git clone https://github.com/inject3r/charlist.git
cd charlist
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npm run deploy
```
